=== Nexus ===
Author: Kanchana Fernando
Requires at least: 5.6
Tested up to: 6.1
Requires PHP: 5.6

== Description ==
Nexus - A WordPress theme designed to showcase your events in an appealing way. This theme is simple theme with minimal design and all the key features required for a WordPress theme. It has responsive layouts that works well on any device.

== Installation ==

1. Log into your WordPress dashboard, go to Appearance -> Themes and click the "Add New" button.
2. Click on "Upload Theme" button and choose the "nexus.zip" file and click on "Install Now".
3. Click on the "Activate" button to use the Nexus theme on your WP site.
4. Navigate to Appearance > Customize in your dashboard to customize the theme as per your need.
