=== Event manager ===
Author: Kanchana Fernando
Requires at least: 5.0
Tested up to: 6.1

== Description ==

<p>This plugin allows to easily display your events on a page/post using a simple shortcode. You can configure the output of the shortcode by passing various parameters to the shortcode using the "Shortcode Generator" subpage of the plugin menu.</p>

<p>This plugin provides five mobile friendly layouts to showcase your events. The default layout is Grid layout. Those layouts are as follows.</p>
	<ul>
		<li>Default</li>
		<li>Grid Layout</li>
		<li>Full Width Layout</li>
		<li>Boxed Layout (Image Left - Content Right)</li>
		<li>Boxed Layout (Content Left - Image Right)</li>
		<li>Content Overlay Layout</li>
	</ul>	

== Installation ==

1. Log into your WordPress dashboard, go to Plugins -> Add New.
2. Click on "Upload Plugin" button at the top of the page.
3. Choose the "event-manager" plugin zip file and click on "Install now".
4. Click on the "Activate Plugin" button to activate and use the plugin.
